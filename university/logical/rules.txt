Если личный кабинет = доступен то студент = учится 
Если личный кабинет = не доступен то студент = отчислен
Если студент = отчислен то университет = закрыт
Если студент = не отчислен то университет = открыт
Если университет = закрыт то поход на пары = не доступен
Если университет = открыт то поход на пары = доступен
Если поход на пары = доступен то получение оценки = возможно
Если поход на пары = не доступен то получение оценки = невозможно
Если получение оценки = возможно то возможность написать экзамен = есть
Если получение оценки = невозможно то возможность написать экзамен = нет
Если возможность написать экзамен = есть то сдача сессии = возможна
Если возможность написать экзамен = нет то сдача сессии = невозможна
